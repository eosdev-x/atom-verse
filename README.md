A simple Bible verse search site.

## Installation

Ensure you have the following prerequisites installed:
- Node

# Clone the repository
```bash
git clone https://github.com/eosdev-x/atom-verse.git
```
# Navigate to project directory
```bash
cd atombin
```
# Run project
```bash
npm i
```
```bash
npm run dev
```
